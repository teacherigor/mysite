Гризіть фразові дієслова тут!

На цій сторінці в зможете зануритися в світ англійських фразових дієслів.

Фразове дієслово (у нас такого нема!) - це певне англійське дієслово (go, get, take, look, etc) плюс певний англійський прийменник (in, on, at, over, etc), які разом утворюють НОВЕ англійське дієслово.
turn sb in - здати когось (або здатися самому) в поліцію
Зразок:

Водій-утікач здався = turned himself in в поліцію на наступний день після аварії.

give up - припинити щось робити (переклад може варіюватися в залежності від змісту)
Зразок:

- Ти ніколи не відгадаєш відповідь. Здаєшься? = Do you give up?

- Я здаюся. = I give up. Скільки їх там було?

- Їй важко переконати керівництво прийняти її пропозицію, але вона рішуче налаштована не поступатися = not to give up

set (sth) up - офіційно заснувати нову компанію, організацію, систему, метод роботи і т.і.
Зразок:

- Комітет було засновано = has been set up, щоб організовувати соціальні події для студентів.

- Вона планує заснувати = to set up свій власний бізнес.

- Вони заснували = have set up фонд для жертв землетрусу.

set up sth - організувати певну подію або діяльність.
Зразок:

- Нам потрібно організувати = to set up зустріч, щоб обговорити пропозиції.

- Уряд погодився організувати = to set up публічне розслідування.

- Вони організували = have set up телефонну лінію допомоги для дітей, які страждають від булінга.
