Гризіть фразові дієслова тут!

На цій сторінці ви зможете зануритися в світ англійських фразових дієслів.

Фразове дієслово (у нас такого нема!) - це певне англійське дієслово (go, get, take, look, etc) плюс певний англійський прийменник (in, on, at, over, etc), які разом утворюють НОВЕ англійське дієслово.
встановити, побудувати, скласти разом певну конструкцію - set up sth
Зразок:

Митці встановлюють = set up мольберти, щоб демонструвати свої картини.

офіційно заснувати, "побудувати" нову компанію, організацію, систему, метод роботи і т.і. - set (sth) up
Зразок:

- Комітет було засновано = has been set up, щоб організовувати соціальні події для студентів.

- Вона планує заснувати = to set up свій власний бізнес.

- Вони заснували = have set up фонд для жертв землетрусу.

організувати певну подію або діяльність - set up sth
Зразки:

- Нам потрібно організувати = to set up зустріч, щоб обговорити пропозиції.

- Уряд погодився організувати = to set up публічне розслідування.

- Вони організували = have set up телефонну лінію допомоги для дітей, які страждають від булінга.

влаштувати(ся) на певну посаду або у певний бізнес - set sb up as/in
Зразки:

- Після того, як він закінчив коледж, його батько влаштував його = set him up у сімейний бізнес.

- Коли він втратив свою роботу, його дядько влаштував його = set him as консультантом в свою компанію.

- Вона влаштувалася = set herself up дизайнером інтер'єру.

забезпечити грошима, які хтось потребує для важливого завдання або діяльності, які очукуються, що триватимуть довгий час - set sb up
Зразок:

Лотерейний виграш забезпечив їх = has set them up на все життя.

забезпечити когось енергією або здоров'ям, які ця людина потребує, на певний період часу - set sb up
Зразок:

Хороший сніданок дійсно забезпечує вас енергією = set you up на цілий день.

організувати зустріч між двома людьми з метою заохочення романтичних стосунків між ними = "зводити" людей - set sb up with sb
Зразок:

- Тодд спробував звести її = set her up з одним зі своїх друзів.

- Друзі Люсі спромоглися познайомити її = set her up з великою кількістю хлопців.

- Я думаю, нам слід познайомити Райана з моє двоюрідною сестрою = set Ryan up with my cousin.

обманути людину для того, щоб примусити її зробити щось або для того, щоб примусити її виглядати винною у тому, чого ця людина не робила = "підставити" когось - set sb up
Зразки:

- Вони стверджували, що вони не продавали наркотики, але що їх "підставила" = they had been set up поліція.

- Якщо Майкл невинний, тоді, його "підставили" = he was set up його ж друзі.

налаштувати когось на певну дію, поведінку - set sb up to do sth
Зразок:

Якщо ми не будемо давати нашим дітям освіту, ми "налаштуємо" їх програти = set them up to fail у цьому житті.
