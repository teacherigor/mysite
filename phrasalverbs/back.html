(про транспортний рух) вставати в довгу лінію, накопичуватися = back up
Зразки:

- Транспортний рух починає вишукуватися в довгу шеренгу = back up на трасі М25.

- На Лондонській трасі М11 рух було розтягнуто = was backed up на декілька миль.

- Машини та інші транспортні засоби вишукувалися на декілька годин, щоб досягти парома, розтягнувшись = backing up по Стоу Сквер аж до Фоссвеі.

здавати назад (про транспорт); рухатися назад (про людину) = back up
Зразки:

- Куме, здайте трохи назад = back up a bit, будь ласка.

- Він здав декілька футів назад = backed up a few feet та зупинив вантажівку.

- Вона відступила назад = backed up декілька кроків.

- Я відступив обережно назад = backed up carefully, поки не відчув за своєю спиною стіну.

підстрахувати когось/щось, надати комусь/чомусь підтримку = back sb/sth up
Зразки:

- Моя сім'я підтримала мене = back me up впродовж судового процесу.

- Генеральний Секретар каже, що декларація наразі повинна бути підтримана = must be backed up конкретними та ефективними діями.

- Вже час, як уряду слід було підтримати = backed up свої рекламні обіцянки більш суворими мірами.
